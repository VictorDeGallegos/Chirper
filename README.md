# Chirper 🐦

A simple micro-blogging application built with **Laravel** as part of the *Getting Started with Laravel* bootcamp by [Josh Cirre](https://github.com/joshcirre).

This project demonstrates the fundamentals of building a modern Laravel web application including authentication, CRUD operations, database migrations, and deployment.

The application allows users to post short messages called **chirps**, edit them, and interact with content in a simple social feed.

---

## 🚀 Features

- User authentication (register, login, logout)
- Create, edit and delete chirps
- Real-time validation and clean UI
- Database migrations and seeders
- MVC architecture following Laravel best practices
- Deployment-ready Laravel configuration

## 📸 Application Overview

Chirper is a minimal social feed where users can:

- Create a chirp
- Edit their chirps
- Delete chirps
- View chirps from other users
- Log in and out
- Register
- Guest users can view and add chirps

The goal of the project is to understand the Laravel workflow including:

- Routing
- Controllers
- Models
- Migrations
- Blade views
- Authentication
